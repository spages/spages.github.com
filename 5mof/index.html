## Warlock Setup Run make and create 3 nodes in the `dev` folder. $ make $ make devrel Start all the nodes $ dev/dev1/bin/warlock start $ dev/dev2/bin/warlock start $ dev/dev3/bin/warlock start Connect two of the nodes to the first one $ dev/dev2/bin/warlock-admin join warlock1@127.0.0.1 $ dev/dev3/bin/warlock-admin join warlock1@127.0.0.1
## Warlock Usage Format 

    war_server:x(LOCATION, COMMAND)
Cluster (clu) 

    (warlock@127.0.0.1)1< war_server:x(clu, [set, key, value]).
    {ok,success}
Local (loc) 

    (warlock@127.0.0.1)> war_server:x(loc, [get, key]).
    {ok,value}
## Warlock Client RPC (Stable) 

    rpc:call(Node, war_server, x, [loc, Cmd])
Redis client (alpha) 

    {ok, C} = eredis:start_link("127.0.0.1", "9736",  
                                "1", "password").
    {ok, <<"OK">>} = eredis:q(C, [clu, [set, foo, bar]]).
